JavaScript

Introducción General

JavaScript es un lenguaje de programación interpretado, de alto nivel y multiparadigma, ampliamente utilizado en el desarrollo web.
Fue creado en 1995 por Brendan Eich con el objetivo de dotar de interactividad a las páginas web.
Hoy en día, JavaScript no solo se usa en navegadores, sino también en el lado del servidor (Node.js), en aplicaciones móviles, de escritorio y videojuegos.
Es uno de los tres pilares fundamentales del desarrollo web, junto con HTML (estructura) y CSS (estilo).

JavaScript

Características Principales

Interpretado

No requiere compilación previa; el navegador ejecuta el código directamente, permitiendo probar y modificar fácilmente sin pasos intermedios.

multiparadigma

Soporta programación imperativa, orientada a objetos y funcional, lo que brinda flexibilidad para adaptarse a distintos estilos de desarrollo.

Dinámico y débilmente tipado

Las variables no necesitan declarar un tipo específico y pueden cambiar de tipo durante la ejecución, ofreciendo libertad pero requiriendo atención para evitar errores.

Basado en eventos

Permite responder a acciones del usuario, como clics o movimientos del mouse, haciendo posible la creación de interfaces interactivas y dinámicas.

Compatible con navegadores

Funciona de forma nativa en todos los navegadores actuales sin necesidad de complementos adicionales, lo que lo convierte en un estándar del desarrollo web.

Extensible

Puede comunicarse con APIs, bases de datos y frameworks, y gracias a entornos como Node.js también se usa fuera del navegador para desarrollar aplicaciones completas.
